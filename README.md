# Philippine Election 2025 Network Science Project

**Author:** Bob Mathew D. Sunga, MSc  
**Project:** From Online Attention to Electoral Outcomes: A Network Science Analysis of Philippine Election 2025 Twitter/X Communication

This repository contains a reproducible network science project that analyzes Philippine Election 2025 Twitter/X communication and compares online network signals with actual Philippine Senate election outcomes.

The repository is prepared for GitHub upload using a clean source-first structure. Raw datasets and generated outputs are intentionally excluded so forkers can run the notebooks themselves.

## What this project does

The project turns Twitter/X election discourse into multiple network structures:

- hashtag co-occurrence network
- candidate co-mention network
- candidate-hashtag bipartite network
- mention/reply communication network
- network metrics compared with actual Senate vote outcomes
- geographic comparison between national and NCR election patterns
- animated storytelling outputs
- Django dashboard for interactive review

## Quick start with Docker

Build the containers:

```bash
docker compose build
```

Check whether the required datasets are present:

```bash
docker compose run --rm jupyter python docker/check_data.py
```

Open JupyterLab:

```bash
docker compose up jupyter
```

Then open:

```text
http://localhost:8888
```

Run the full notebook pipeline:

```bash
docker compose run --rm jupyter bash docker/run_all_notebooks.sh
```

Start the Django dashboard after the notebooks finish:

```bash
docker compose up dashboard
```

Then open:

```text
http://localhost:8000
```

## Important interpretation

This project does not claim that Twitter/X directly predicts elections. It studies whether online political attention and network centrality align with actual election outcomes. The results are best interpreted as a comparison between online visibility structures and electoral outcomes, not as a causal election prediction model.
