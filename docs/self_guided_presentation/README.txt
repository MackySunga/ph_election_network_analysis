Philippine Election 2025 Network Science — Self-Guided HTML Presentation App
Prepared by Bob Mathew D. Sunga, MSc

How to run:
1. Open index.html directly in a browser, or
2. Run a local server:
   python -m http.server 8000
   then open http://localhost:8000

Controls:
- Right arrow / Space: next slide
- Left arrow: previous slide
- M: toggle slide menu

This V4 version includes expanded explanatory text on each slide so the app can be uploaded to a web server and understood even without a live presenter.
